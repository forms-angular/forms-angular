In this really simple set-up there are a few routes for each model (and if you have followed the quick-start you will have a model called applicant so the first two links will work).

The routes are:

  • /#/:model which lists the documents in the collection, with links to the form for editing them.
  • /#/:model/new which enables the user to create a new document using the default form for the collection.
  • /#/:model/:id/edit which enables the user to edit a document

TYou might also want to try the search box in the navbar (once you have added some data).
